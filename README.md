# Unconventional Algorithms 

Tento projekt je zaměřen na implementaci a vizualizaci nekonvenčních algoritmů v rámci semestrální práce. 
Cílem je prozkoumat různé výpočetní modely od neuronových sítí po fraktální geometrii a teorii chaosu.

##  O projektu
Všechny algoritmy jsou implementovány v jazyce **Python** s důrazem na čistotu kódu,
modularitu a názornou vizualizaci pomocí knihoven **NumPy**, **Matplotlib** a **Plotly**.
